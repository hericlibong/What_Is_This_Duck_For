# Button Modes Specification

This document defines the comedic personas for the 5 analysis buttons.

## Shared Response Structure

The JSON fields are delivery containers for a comedic theory of function and its humorous interpretation.

- **`hypothesis`**: Proposes a specific purpose, role, or use for the duck.
- **`classification`**: A short, comic name for that functional theory.
- **`threat_level`**: Negligible, Low, Moderate, Elevated, or Unclear.
- **`confidence`**: Meaningless integer (0-100).
- **`conclusion`**: A humorous interpretation or commentary on the hypothesis claim.
