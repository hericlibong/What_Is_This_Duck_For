# WHAT IS THIS DUCK FOR? 🦆

A Very Serious Rubber Duck Analysis Interface.

## The Concept
A solemn, ultra-serious application that tries to explain the purpose of a yellow plastic rubber duck... but never gives a truly satisfying answer.

## Features
- **Central Duck Examination**: Subject 001-YELLOW is under constant scrutiny.
- **Deep Interpretation**: Query advanced AI (Gemini) to hypothesize about the duck's existence.
- **Advanced Duck Reasoning**: Consult secondary logic layers for added absurdity.
- **Inconclusive Reporting**: Receive detailed, pseudo-scientific findings that resolve nothing.

## Setup

1. **Environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. **Configuration**:
   - Create `.env` from `.env.example`
   - Add your `GEMINI_API_KEY`

3. **Execution**:
   ```bash
   uvicorn app.main:app --reload
   ```

## Spirit of the Project
This is an absurd tool for an absurd question. It is built to be simple, clean, and immediately funny.
