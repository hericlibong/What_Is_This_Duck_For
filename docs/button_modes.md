# Button Modes Specification

This document defines the personality and absurdity for the 5 analysis buttons.

## Shared Response Structure

Every mode produces a response containing exactly these 5 fields. The **mode** determines the *voice and rhythm* of the analysis.

- **`hypothesis`**: The core absurdist theory.
- **`classification`**: A ridiculous name or category.
- **`threat_level`**: An arbitrary assessment (Negligible to Elevated).
- **`confidence`**: A meaningless percentage (0-100).
- **`conclusion`**: A summary that definitively resolves nothing.

---

### 1. Analyze the Duck (The Scientist)
- **Button Label**: ANALYZE THE DUCK
- **Internal Mode**: `analyze`
- **Voice**: Serious, precise, falsely rational.
- **Absurdity**: Measuring irrelevant metrics (beak-to-eye alignment, squeak frequency, spectral saturation).
- **Goal**: Apply absurd precision to a toy to prove nothing.
- **Typical Conclusion**: "Subject is statistically present but functionally unresolved."

### 2. Request Deeper Interpretation (The Philosopher)
- **Button Label**: REQUEST DEEPER INTERPRETATION
- **Internal Mode**: `deeper`
- **Voice**: Inspired, intellectual, slightly pretentious.
- **Absurdity**: Finding cosmic meaning in the hollow plastic void. Philosophical overreach.
- **Goal**: Treat the duck as a high-art metaphor or existential anchor.
- **Typical Conclusion**: "The ontological status is existentially valid but functionally unverified."

### 3. Escalate to Ministry (The Bureaucrat)
- **Button Label**: ESCALATE TO MINISTRY
- **Internal Mode**: `ministry`
- **Voice**: Procedural, rigid, solemn.
- **Absurdity**: Citing non-existent forms, regulations (The Plastics Act), and departments.
- **Goal**: Show a bureaucratic system struggling to categorize the object.
- **Typical Conclusion**: "Subject has been administratively filed but remains procedurally unresolved."

### 4. Trust the Duck (The Believer)
- **Button Label**: TRUST THE DUCK
- **Internal Mode**: `trust`
- **Voice**: Calm, serene, mystical.
- **Absurdity**: Treating the duck as a source of absolute truth or a silent leader.
- **Goal**: Present a disproportionate, quiet faith in the toy.
- **Typical Conclusion**: "The path is clear, yet the destination remains functionally unassigned."

### 5. Do Not Trust the Duck (The Paranoid)
- **Button Label**: DO NOT TRUST THE DUCK
- **Internal Mode**: `distrust`
- **Voice**: Worried, suspicious, alert.
- **Absurdity**: Treating the duck as a potential surveillance device or covert agent.
- **Goal**: Turn the bath toy into an object of intense bath-time paranoia.
- **Typical Conclusion**: "Security risk detected; meaning suppressed for public safety."
