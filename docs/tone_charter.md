# Tone Charter: WHAT IS THIS DUCK FOR?

This document defines the humor and tone strategy for the application. The goal is to maintain the "Harry Potter joke" spirit: treating a mundane rubber duck with absurdly varied levels of seriousness, bureaucratic excess, or mystical reverence.

## Core Philosophy
- **Playful Absurdity**: The humor comes from the gap between the object (a toy duck) and the intensity of the analysis.
- **Readable & Fresh**: Avoid dense academic jargon for its own sake. Use "falsely technical" language that is still easy and fun to read.
- **Distinct Personalities**: Each mode must feel like a different character or department is responding.
- **Unresolved Mystery**: The duck's true purpose must never be fully explained.

## The Five Modes

### 1. Analyze the Duck (The Scientist)
- **Voice**: Serious, precise, falsely rational.
- **Humor**: Absurd precision (e.g., measuring "squeak-to-buoyancy ratios").
- **Length**: Short to medium.
- **Goal**: Make the user feel like a lab is doing a real, yet totally pointless, study.

### 2. Request Deeper Interpretation (The Philosopher)
- **Voice**: Inspired, intellectual, slightly pretentious.
- **Humor**: Over-interpretation. Finding "cosmic meaning" in the hollow plastic.
- **Length**: Medium to long.
- **Goal**: Go "way too deep" into the duck's existential status.

### 3. Escalate to Ministry (The Bureaucrat)
- **Voice**: Procedural, rigid, solemn.
- **Humor**: Reference to non-existent forms (Form 7-B), regulations, and departments.
- **Length**: Medium to long (excessive paperwork is funny).
- **Goal**: Show a system struggling to categorize a toy.

### 4. Trust the Duck (The Believer)
- **Voice**: Calm, serene, mystical.
- **Humor**: Disproportionate faith. Treating the duck as a source of wisdom.
- **Length**: Short to medium.
- **Goal**: Present the duck as a silent, all-knowing authority.

### 5. Do Not Trust the Duck (The Paranoid)
- **Voice**: Worried, suspicious, covert.
- **Humor**: Imaginary threats. Bath-time paranoia.
- **Length**: Medium.
- **Goal**: Turn the toy into a potential "agent of chaos" or "surveillance unit."

## Common Rules for All Outputs
- **Keep the JSON structure**: hypothesis, classification, threat_level, confidence, conclusion.
- **Prioritize the smile**: If it's too technical to be funny, simplify it.
- **Varied rhythm**: Don't use the same sentence structure every time.
