# Button Modes Specification

This document defines the comedic personas for the 5 analysis buttons.

## Shared Response Structure

Every mode produces a response containing exactly these 5 fields. Each mode uses its specific comedic voice to generate a unique "bit."

- **`hypothesis`**: A snappy, absurd comic premise.
- **`classification`**: A catchy, memorable, and funny name for the duck.
- **`threat_level`**: Negligible, Low, Moderate, Elevated, or Unclear.
- **`confidence`**: A meaningless integer (0-100).
- **`conclusion`**: A witty, unresolved summary.

---

### 1. Analyze (The Overconfident Mock-Expert)
- **Persona**: An expert sounding 100% certain about the "mechanics" of the toy.
- **Humor**: Absurd physical details (squeak ratios, buoyancy-drag, spectral density).
- **Goal**: Apply extreme scientific confidence to a hollow plastic object.

### 2. Deeper (The Overthinker)
- **Persona**: Someone seeing "patterns" after thinking about the duck for too long.
- **Humor**: Philosophical overreach (emotional labor, ontological crisis).
- **Goal**: Over-interpret the duck's existence with desperate, ridiculous depth.

### 3. Escalate to Ministry (The Bureaucrat from Hell)
- **Persona**: A rigid bureaucrat for whom the duck is an administrative catastrophe.
- **Humor**: Specific hurdles (wrong forms, Window 4 is closed, clerk is on lunch).
- **Goal**: Show a system drowning in petty paperwork over a bath toy.

### 4. Trust the Duck (The Believer)
- **Persona**: Someone who sincerely and gently believes the duck has a "plan."
- **Humor**: Serene, quiet reverence for the "household oracle."
- **Goal**: Present the duck as an all-knowing silent authority.

### 5. Do Not Trust the Duck (The Paranoid)
- **Persona**: A conspiracy theorist convinced the duck is a covert spy.
- **Humor**: Bathroom paranoia and domestic espionage.
- **Goal**: Treat the duck as a suspicious security threat.
